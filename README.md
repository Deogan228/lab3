Разработайте и реализуйте объектно-ориентированную модель,
рассчитывающую общую стоимость и вес покупки в магазине. Сама покупка
собирается из различных товаров и продуктов, которые покупает отдельный
человек. Известно, что все товары и продукты обладают своей стоимостью и
весом. Предусмотрите возможность добавления новых общих свойств. 

Item.scala — ADT для товаров: один sealed trait, три case class.

Purchase.scala — логика подсчёта стоимости/веса/объёма.

Input.scala — ленивый поток ввода LazyList[String].

ConsoleAsk.scala — функциональный ввод с валидацией и рекурсией.

Main.scala — сборка всей движухи: читаем → создаём Purchase → считаем → выводим.
